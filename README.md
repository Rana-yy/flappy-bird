# Flappy Bird Demo

A Flappy Bird clone project developed using Java, JavaFX, and object-oriented programming principles.

---

## 📌 Project Description
This project is a simple Flappy Bird style game developed with JavaFX.  
The player controls a bird, avoids pipes, and tries to achieve the highest score possible.

The project was created to improve Java programming, JavaFX usage, and object-oriented programming (OOP) skills.

---

## 🛠 Technologies Used
- Java
- JavaFX
- Object-Oriented Programming (OOP)
- Git & GitHub

---

## ▶️ How to Run
1. Open the project in IntelliJ IDEA
2. Make sure JavaFX installed
3. Run Main.java

---

## 🎮 Game Features
- Gravity system
- Bird movement mechanics
- Random pipe generation
- Collision detection
- Score system
- Game over screen


---

## ⭐ Extra Features
- Restart 
- Login screen
- Secure local authentication (Login & Register via User Storage)
- Local high score persistence using `RandomAccessFile`
- Seamless in-place restart mechanism
- Collision management utilizing AABB algorithms
